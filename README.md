# frontend-task

A password-protected page for Innova's web developer hiring task.

- Built as a single-page site with [Astro](https://astro.build).
- Encrypted with [StatiCrypt](https://github.com/robinmoisson/staticrypt) (client-side AES).
- Served via GitHub Pages.

If you were sent here, you'll have received the access password separately.
